# Laravel Project Management App

## Features
- Auth via Laravel Breeze
- Projects & tasks CRUD
- Task filtering
- Dashboard metrics
- Blade frontend

## Setup
1. Clone repo
2. `composer install`
3. `npm install && npm run dev`
4. Setup `.env`
5. `php artisan migrate`
6. `php artisan serve`

## Tech Stack
- Laravel 12
- Blade
- Breeze (Auth)
- MySQL
